1)Третий режим ("оффлайн"): программе подается входной файл, количество итераций, выходной файл. Программа рассчитывает состояние поля после указанного количества итераций и сохраняет вселенную в указанный выходной файл.
Пример:
C:\Users\vipko\source\repos\nsu_OOP_lab2\x64\Debug\nsu_OOP_lab2.exe -i 15 -o C:\Users\vipko\source\repos\nsu_OOP_lab2\res2.txt -in C:\Users\vipko\source\repos\nsu_OOP_lab2\Board.txt

-i x  - количество итераций, -o path to the file - выходной файл.
-in path to the file - входной файл

2)Формат файла:

#Name_of_universe Bx/Sy
1 2
3 1
2 2 
2 4
3 7
7 13

Число x под буквой B (birth) содержит количества живых ячеек, необходимых для "рождения" клетки, последовательность цифр под буквой S (survival) - ячеек, необходимых для "выживания" клетки
